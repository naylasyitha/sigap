**Deskripsi SIGAP**

SIGAP (Sistem Informasi Gizi dan Pemantauan Anak) adalah aplikasi berbasis web yang dirancang untuk membantu orang tua dalam memantau tumbuh kembang anak secara mudah, praktis, dan terstruktur. Aplikasi ini menyediakan fitur pencatatan data pertumbuhan anak, visualisasi grafik pertumbuhan berdasarkan standar WHO, pengelolaan jadwal kesehatan anak seperti imunisasi dan penimbangan, serta rekomendasi menu MPASI sesuai usia anak.

Aplikasi ini dikembangkan sebagai proyek tugas akhir Mata Kuliah Pengembangan Aplikasi Web dengan tujuan menerapkan konsep pengembangan aplikasi web modern, mulai dari perancangan antarmuka pengguna, integrasi frontend dan backend, hingga pengelolaan basis data dengan mengangkat studi kasus yang relevan dengan kebutuhan masyarakat.
